1️⃣ What is the difference between var, let, and const?
All the varible have to concern respect to scope and reassign, from this point of view
var = can be both functional scope or global scope depend on declared position, can be reassign a value
let = is block-scope and would not allow to re-declare but allow to reassign a value 
const = block scope like let,  cannot be reassigned, redeclare


2️⃣ What is the spread operator (...)?
... = is a spread operator its spread the element of array for example
const a = [1,2,3]
const b = [...a, 4,5] 
b = [1,2,3,4,5]


3️⃣ What is the difference between map(), filter(), and forEach()?
The map() method creates a new array by performing a function on each array element.

The map() method does not execute the function for array elements without values.

The map() method does not change the original array.

filter () will create a new array of the array element which will pass througha  test

forEach() will not return any array but a function will be called for each array element
